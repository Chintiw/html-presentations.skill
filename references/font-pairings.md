# Font Pairings for Presentations

All pairings use Google Fonts. Load via `<link>` in `<head>`.

---

## Executive / Investor Pitch

**Display**: Cormorant Garamond (600, 700 italic)  
**Body**: DM Sans (300, 400, 500)

```html
<link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,600;0,700;1,600;1,700&family=DM+Sans:wght@300;400;500&display=swap" rel="stylesheet">
```
```css
--font-display: 'Cormorant Garamond', serif;
--font-body: 'DM Sans', sans-serif;
```
*Palette suggestion*: Near-black `#0d0d0d`, warm white `#f4f0e8`, gold `#c9a84c`

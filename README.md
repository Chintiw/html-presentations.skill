**`html-presentations.skill`** — install this to give Claude the ability to build stunning browser-based slide decks.

---

### What the skill does

When triggered (e.g. *"make me a pitch deck"*, *"create slides about X"*, *"I need a presentation"*), Claude will:

1. **Understand the content** — extract topic, audience, tone, and length, or make smart assumptions and proceed
2. **Choose a design archetype** — from Executive Dark to Playful Studio, committing to one bold visual direction with a memorable signature element
3. **Build a single self-contained `.html` file** with embedded CSS and JS — no dependencies except Google Fonts

### What's inside the skill package

| File | Purpose |
|---|---|
| `SKILL.md` | Core instructions: architecture, navigation, animation, slide types, output rules |
| `references/slide-patterns.md` | Copy-paste HTML+CSS for 8 slide types (title, statement, stats, quote, steps, divider, split, closing) |
| `references/font-pairings.md` | 7 curated Google Fonts pairings by aesthetic, with palette suggestions |
| `references/design-principles.md` | Design theory: hierarchy, contrast, motion, negative space, pacing |

### The output every time
A single `.html` file with arrow-key + click + swipe navigation, animated slide entrances, a progress bar, and a slide counter — beautiful enough to present directly from a browser tab.
